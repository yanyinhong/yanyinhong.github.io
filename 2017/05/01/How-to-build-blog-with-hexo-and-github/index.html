部署到GitHub Pages

前面有说过，GitHub Pages用于介绍托管在GitHub的项目的，每个帐号只能创建一个repository来存放GitHub Pages，而且仓库名称必须是username/username.github.io，这是固定的命名约定。

创建后，可以通过http://username.github.io来访问个人主页。

GitHub Pages中个人主页的内容是在master分支下的。
