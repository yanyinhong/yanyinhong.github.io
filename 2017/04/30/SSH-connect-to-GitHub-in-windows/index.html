设置账户

打开系统cmd。设置Git的user name和email。

  • 查看ssh密钥

查看是否已经有了ssh密钥。

默认.ssh文件夹会在用户文件夹中生成。可以前往该路径查看。如果没有该文件夹，终端会提示系统找不到指定的路径,有的话可以备份删除。

%USERPROFILE%是环境变量，表示当前用户文件夹路径。

因为%USERPROFILE%一般在C盘，从其他盘无法直接cd到该路径，所以需要先切换到c盘。
