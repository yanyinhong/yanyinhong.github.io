Mac下安装npm全局包提示权限不够

11月 15, 2017 发布在 前端

Mac OS下安装npm的全局包，总是出现如下提示Missing write access，需要提升权限才能继续。

解决方法1

使用sudo。在安装命令前加上sudo,输入用户的登陆密码，提升权限进行安装。

解决方法2

修改usr/local的权限。使用sudo有一个风险是安装包可能会运行自己的一些脚本，使sudo操作变的不可控，不安全。

可以通过将/usr/local的own从root转为登陆用户来解决这个问题。
