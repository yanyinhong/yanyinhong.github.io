好用的Atom插件（持续更新中）

7月 25, 2017 发布在 前端

Atom提供了大量丰富好用的插件。下面安利一下使用中好用又有趣的插件。

列表

  • terminal-plus
  • platformio-ide-terminal
  • vim-mode
  • ex-mode
  • relative-numbers
  • pretty-json
  • atom-beautify
  • file-icons
  • minimap
  • activate-power-mode

介绍

terminal-plus

terminal-plus支持将终端集成到编辑器中。在编辑器界面就可以打开终端执行命令，省去了另开窗口的麻烦。打开终端时，路径会定位到编辑器项目中的根目录，可以非常方便的对当前项目进行命令操作。

platformio-ide-terminal

platformio-ide-terminal在terminal-plus的基础之上集成了PlatformIO IDE。

vim-mode

vim-mode支持完整的vim模式，在Atom编辑器中也可以使用vim的编辑方式。

ex-mode

ex-mode结合vim-mode使用，支持编辑器的vim模式下的:w:s等命令。

relative-numbers

relative-numbers结合vim-mode使用，实现常规模式下的相对行号。

pretty-json

pretty-json提供json文件的格式化功能。

atom-beautify

atom-beautify提供代码的格式化功能，支持格式化HTML, CSS, JavaScript, PHP, Python, Ruby, Java, C, C++, C#, Objective-C, CoffeeScript, TypeScript, Coldfusion, SQL等等。

file-icons

file-icons提供文件后缀相关的带颜色icon，放置在文件名前面。从视觉上更直观的区分文件类型。

minimap

minimap提供代码缩略图，可以放在右侧用于代码概览和导航。

activate-power-mode

activate-power-mode提供非常酷炫的代码编写效果！
